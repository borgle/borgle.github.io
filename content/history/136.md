```js
function clear() {
    Source=document.body.firstChild.data;
    document.open();
    document.close();
    document.body.innerHTML=Source;
}
```

把这个代码放到你的网页中把，让其`onload`，你会发觉点击“查看源代码”为空，这个东西真算有想法！
